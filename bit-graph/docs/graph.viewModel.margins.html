margins

  • property
graph.viewModel.margins

{Array}

 
  • source

Margins between the graph and its containing element. Should be a four-length array similar to a CSS margin property ([margin-top, margin-right, margin-bottom, margin-left])

Array

Defaults to [20, 0, 20, 80].
